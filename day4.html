With relevant examples, explain the difference between the three ways you can identify string in JS.

Single-quoted Strings(''): It is the easiest way to define a string. It can be used it when you want the string to be exactly as it is written. All the escape sequences like will be output as specified instead of having any special meaning. Single-quote is usually faster in some cases.

Double-quoted strings(""): By using Double quotes the code is forced to evaluate the whole string. The main difference between double quotes and single quotes is that by using double quotes, you can include variables directly within the string. It interprets the Escape sequences. Each variable will be replaced by its value.

Backslash(``) The backslash indicates that the next character should be treated as a literal character rather than as a special character or string delimiter.
